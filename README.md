# Design TCS FPGA
## Лабораторные работы по курсу «Проектирование телекомммуникационных систем на ПЛИС»

В данном репозитории представлены лабораторные работы, выполненные в рамках курса «Проектирование телекомммуникационных систем на ПЛИС».

Каждая лабораторная работа включает:
- исходный код;
- результаты моделирования;
- отчёт с описанием выполненных действий и полученных результатов.
