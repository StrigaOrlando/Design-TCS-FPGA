# Design TCS FPGA
## Лабораторные работы по курсу «Системное проектирование в телекоммуникациях»

В данном репозитории представлены лабораторные работы, выполненные в рамках курса «Системное проектирование в телекоммуникациях».

Каждая лабораторная работа включает:
- исходный код;
- результаты моделирования;
- отчёт с описанием выполненных действий и полученных результатов.
